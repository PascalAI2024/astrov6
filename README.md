## 📦 Installation

1. Clone the repository
2. Install dependencies:
```bash
npm install
```
3. Create `.env` file with required keys:
```env
OPENAI_API_KEY=your-key
NASA_API_KEY=your-key
DATABASE_URL=your-url
CRON_SECRET=your-secret
```

## 🚀 Development

Start the development server:
```bash
npm run dev
```
